Detailed Description

Test the Arduino Button Library Extension (ABLE).

This program will periodically output "Looping..." to the Serial port. As you press buttons connected to pins 2 and 3 it will assert the button settings in the callback fuctions and in the main loop.

The TestAbleButton program is split (by design) into multiple files to ensure ABLE works in multi-module programs. A common TestAbleButton.h header file declares common elements shared across these modules.
