Detailed Description

template<typename ButtonType = class PullupButtonType>
class jsware::BasicButton< ButtonType >

Simplest button class to check if the button has been pressed.

Parameters
ButtonTypeEither a PullupButtonType or PulldownButtonType matching the resistor configuration of the button. Will default to PullupButtonType as the simplest button connection.
