void jsware::BaseButtonType::buttonDebounce ( uint8_t  debounceTime)
staticinherited

Set the debounce time for buttons.

Buttons may generate noise as they are pressed and released. This noise can be registered as multiple button presses in a short time period, confusing a program. The debounce time is the number of milliseconds to wait when a button state changes before returning that change with the isPressed() methods. See https://docs.arduino.cc/built-in-examples/digital/Debounce for details on debounce.

Parameters
debounceTimeThe number of milliseconds before a button state change is returned (max 255ms).
